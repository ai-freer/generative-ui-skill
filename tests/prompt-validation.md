## 验证流程

1. 启用 Skill（注入 system prompt + 对应指南模块）
2. 发送测试 prompt
3. 检查模型输出是否包含正确格式的 `show-widget` 围栏
4. 提取 `widget_code`，在浏览器中打开验证渲染效果
5. 切换深色模式，确认视觉兼容
6. 点击可交互元素，确认 `window.__widgetSendMessage()` 调用正确
7. 检查浏览器控制台无 CSP 违规报错
